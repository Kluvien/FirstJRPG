# FirstJRPG - The Last Debugger

A short 2D turn-based RPG prototype made in Unity for a programmer pre-interview test.

## Concept

The player controls Debugger, a system character who must repair a corrupted game world by helping an NPC, collecting a Patch Crystal, and fighting a Bug Beast in a JRPG-style turn-based battle.

## Features

- WASD / Arrow Key movement
- Space interaction
- NPC dialogue
- In-game cutscene
- Turn-based battle system
- ScriptableObject-based character and skill data
- Prefab-based player, NPC, enemy, and interactable objects

## Controls

- Move: WASD / Arrow Key
- Interact: Space
- Select Battle Command: Mouse / Keyboard
